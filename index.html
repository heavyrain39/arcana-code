최신 릴리즈: v1.0

아르카나 코드란?

깔끔한 무료 타로카드 프로그램을 찾고 계셨나요? 여기 〈아르카나 코드〉가 있습니다.

78장의 타로카드 중 랜덤으로 뽑은 세 장의 카드는 각각 과거, 현재, 미래를 상징합니다. 이 세 장의 의미를 구글의 최신 Gemini AI 모델이 풀이해 줍니다. 오늘의 운세, 현재의 애정운, 금전운... 어떤 질문이라도 AI가 모두 알아서 해석해줄 것입니다. 그저 자리에 앉아 긴장을 풀고 질문을 입력하기만 하면 됩니다.

주요 특징

  • 간편한 3카드 스프레드: 과거, 현재, 미래가 한 눈에
  • 로컬 실행: 설치, 가입, 결제 없이 파일 클릭으로 내 PC에서 실행
  • 사용자 API 키 사용: `key.txt`에 본인의 키를 입력해 안전한 사용 가능
  • Gemini AI 해석: 최신 AI가 제공하는 친절하고 상세한 리딩
  • 최적화된 사용자 경험: 다크 테마의 깔끔하고 직관적인 유저 인터페이스

스크린샷

지금 바로 시작하세요!

간단 사용법:

  1. 아래 버튼으로 .zip 파일을 다운로드하세요.
  2. 원하는 곳에 압축을 해제하세요 (폴더가 생성됩니다).
  3. 폴더 안의 `key.txt` 파일을 열어 자신의 Gemini API 키를 입력하고 저장하세요.
  4. `start.bat` 파일을 더블클릭하여 실행하면 끝! (자동으로 브라우저가 열립니다)

개발자 정보

안녕하세요? 바이브 코딩하는 야차완입니다. LLM은 진정한 무작위 결과를 생성하기 어려워서, 타로 카드를 랜덤으로 선택하라고 지침을 주어도 특정 카드가 편향되게 나타난다는 구조적인 문제가 있습니다. 그래서 카드 선택은 자바스크립트에 맡기고, 의미 해석만 LLM이 하는 식으로 역할을 분담하는 앱을 구상했습니다. 이 프로그램은 그 결과물입니다. 유용하게 써주시면 기쁘겠습니다. 피드백이나 문의사항은 아래 트위터(X) 계정으로 연락 부탁드립니다.
